# Практична робота: Автоматизоване тестування та контроль версій

##  Опис проекту
Цей проект демонструє базові навички розробки на Python з використанням модульного підходу та систем контролю версій. 
В основі лежить створення математичного модуля та його валідація за допомогою Unit-тестів.

##  Структура репозиторію
*   `app.py` — основний модуль з логікою (функція множення).
*   `test_app.py` — набір автоматизованих тестів для перевірки коректності обчислень.
*   `README.md` — звіт та документація до проекту.

##  Використані технології
*   **Мова програмування:** Python 3.
*   **Бібліотеки:** `unittest` (стандартна бібліотека для тестування).
*   **Інструменти:** Git, GitHub, macOS Terminal (Zsh).
