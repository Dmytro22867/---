 Практична робота: Автоматизоване тестування та контроль версій

Опис проекту

Цей проект демонструє створення простого інтерактивного калькулятора на Python з використанням модульного підходу та систем контролю версій. Калькулятор підтримує базові математичні операції: додавання, віднімання, множення, ділення, а також дужки та числа з плаваючою комою.

Структура репозиторію
*   `app.py` - основний модуль калькулятора з функцією обчислення виразів.
*   `test_app.py` - набір автоматизованих тестів для перевірки коректності обчислень.
*   `README.md` - звіт та документація до проекту.

Використані технології
*   **Мова програмування:** Python 3.
*   **Бібліотеки:** `unittest` (стандартна бібліотека для тестування), `re` (для валідації виразів).
*   **Інструменти:** Git, GitHub, macOS Terminal (Zsh).

Як запустити калькулятор

Для запуску калькулятора виконайте команду в терміналі:
```
python3 app.py
```
Після цього введіть математичні вирази, такі як `2 + 3 * 4`, і отримайте результат. Введіть `exit` для виходу.

Як запустити тести

Для перевірки працездатності коду необхідно виконати команду в терміналі:
```bash
python3 test_app.py
```
